# CobotInt

Projekt interfejsu webowego do sterowania robotem KUKA LBR iiwa za pomocą React i ROS 2.  
Aplikacja komunikuje się z systemem ROS 2 (lbr_fri_ros2_stack) i umożliwia sterowanie 7-przegubowym robotem z poziomu przeglądarki.  
Projekt w fazie prototypu – środowisko uruchomione lokalnie (VM z ROS 2 Humble).

**Technologie:** React, ROS 2 Humble, WebSocket 
**Status:** Prototyp (lokalne środowisko testowe)
